# locotown/yomikata tests ベースの差分整理（更新版）

- 参照元: https://github.com/locotown/yomikata/tree/main/tests
- 比較日: 2026-02-21
- 比較対象実装: `japanese_number_reading`（現行）
- 比較方法:
  - `tests/counters.test.ts`, `tests/datetime.test.ts`, `tests/numbers.test.ts`, `tests/ordinals.test.ts`, `tests/integration.test.ts` から文字列リテラルケースを抽出
  - 抽出ケースを現行の `replaceInText` で照合
  - 最新結果: 216ケース中 18ケース不一致

## 今回修正して解消した項目

- `位`（`1位`, `2位`, `10位`, `1位を獲得`）
- `第X章` の促音（`第1章`, `第8章`, `第10章`）
- `11歳`（`じゅういっさい`）
- `3箱`（`さんばこ`）
- `1000メートル`（`せんめーとる`）

## 残差分（18件）

### 1. 曜日語彙（8件）

- `月曜日`〜`日曜日`
- `月曜日の9時` のような混在文

### 2. 円の扱いポリシー差（3件）

- locotown: 数字だけ読み化し `円` は漢字のまま（例: `さんびゃく円`）
- 現行: `円` まで読み化（例: `さんびゃくえん`）

### 3. 裸数字の低値ポリシー差（2件）

- locotown: `50`, `99` は未変換
- 現行: 裸数字は全て読み化（`ごじゅう`, `きゅうじゅうきゅう`）

### 4. 固有語彙辞書の差分（5件）

- 例: `豊くん -> ゆたかくん`, `花子さん -> はなこさん`, `今日は -> こんにちは`
- 現行スコープ外（辞書置換未実装）

### 5. 年齢高値のポリシー差（1件）

- locotown: `100歳` は未変換
- 現行: `100歳` も読み化（`ひゃくさい`）
